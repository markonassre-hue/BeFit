# BeFit
A front-end gym web application developed as a college project using HTML, CSS, Tailwind CSS, and JavaScript.

## Project Overview
BeFit is a fitness-focused web application designed to help users track their fitness journey. It includes interactive features, a responsive design, and a user-friendly interface. 

## Key Features
- Responsive Navigation Bar.
- Interactive "Body Transformation" hover effects.
- Service offerings and Pricing plans.
- Login and Registration pages.

## Technologies Used
- HTML5
- CSS3
- Tailwind CSS
- JavaScript
